# 📊 Data Analytics Project

## Overview

This project demonstrates a complete end-to-end data analytics workflow, from raw data to business insights. It covers data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, interactive dashboard creation, and presentation of findings.

The objective is to transform raw data into meaningful insights using industry-standard analytics tools.

---

## Dataset

The dataset contains structured data used for analysis and visualization.

**Key tasks performed:**

* Loaded the dataset using Python
* Explored data structure and statistics
* Identified missing values and duplicates
* Cleaned and prepared the data for analysis

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Examined data types, dimensions, and summary statistics.

### 2. Exploratory Data Analysis (EDA)

* Analyzed distributions and relationships.
* Identified trends, outliers, and missing values.
* Created visualizations for better understanding.

### 3. Data Cleaning

* Removed duplicates.
* Handled missing values.
* Corrected data types.
* Prepared the dataset for SQL analysis.

### 4. SQL Analysis

* Imported the cleaned dataset into MySQL.
* Wrote SQL queries to answer business questions.
* Used filtering, grouping, joins, sorting, and aggregation.

### 5. Power BI Dashboard

* Connected Power BI to the processed data.
* Built an interactive dashboard with KPIs and charts.
* Added filters and slicers for dynamic analysis.

### 6. Reporting & Presentation

* Documented key findings in a report.
* Created a professional presentation using Gamma to summarize insights and recommendations.

## How to Run

1. Clone this repository.
2. Install the required Python libraries.
3. Open the Jupyter Notebook and run all cells.
4. Import the cleaned dataset into MySQL.
5. Execute the SQL queries.
6. Open the Power BI (.pbix) file to explore the dashboard.
7. Review the project report and Gamma presentation for detailed insights.
